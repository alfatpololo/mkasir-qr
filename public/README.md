# Public Folder

Folder ini berisi file-file statis yang bisa diakses langsung dari browser.

## Logo

Taruh file logo Anda di sini dengan nama:
- `logo.png` atau
- `logo.jpg` atau  
- `images/logo.png`

Komponen Logo akan otomatis mencari file di `/logo.png` (root public folder).

Format yang didukung:
- PNG
- JPG/JPEG
- SVG
- WebP

## Struktur Folder

```
public/
├── logo.png          # Logo utama (disarankan)
├── images/
│   └── logo.png     # Alternatif lokasi
└── favicon.ico       # Favicon (opsional)
```
