# Install Dependencies untuk Fitur Export/Import

Untuk menggunakan fitur export/import dan laporan, install dependencies berikut:

```bash
npm install xlsx jspdf jspdf-autotable file-saver date-fns
```

Atau jika menggunakan yarn:

```bash
yarn add xlsx jspdf jspdf-autotable file-saver date-fns
```

## Dependencies yang diperlukan:

1. **xlsx** - Untuk export/import Excel
2. **jspdf** - Untuk generate PDF
3. **jspdf-autotable** - Plugin untuk tabel di PDF
4. **file-saver** - Untuk download file
5. **date-fns** - Untuk format tanggal

Setelah install, semua fitur export/import akan berfungsi dengan baik.
